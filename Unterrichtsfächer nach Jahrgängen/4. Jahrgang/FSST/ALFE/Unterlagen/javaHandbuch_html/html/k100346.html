53.14.1 Aufruf 

jvisualvm [ options ]

53.14.2 Optionen

Option Bedeutung
--openpid processid Visual VM verbindet sich nach dem Start mit einer JVM mit der angegebenen Prozess-Id.
--openfile datei Öffnet die angegebene HPROF-Datei. Sie können auch einen Application-Snapshot übergeben, der mit einer früheren Sitzung erzeugt wurde.

Tabelle 53.13: Optionen von jvisualvm

53.14.3 Beschreibung

Visual VM ist ein Profiler mit zeitgemäßer grafischer Oberfläche. Mit Visual VM können Sie Speicherlöcher aufspüren und Codestellen ausfindig machen, die überproportional viel Laufzeit benötigen. In Abschnitt 52.3.3 führen wir eine Beispielsitzung mit dem Profiler durch.
