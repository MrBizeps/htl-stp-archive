53.15 xjc - der XML Schema Definition Compiler

53.15.1 Aufruf 

xjc [ options ] ( schema file | directory )

53.15.2 Optionen

Option Bedeutung
-d dir Der generierte Quellcode wird in dem angegebenen Verzeichnis abgelegt.
-p paketname Bestimmt den Namen des Pakets der generierten Klassen.
-classpath path Gibt die Liste der Pfade zur Suche von Klassendateien an.
-xmlschema Dieser Parameter weist den Binding Compiler an, die übergebene Datei als eine XML Schema Definition zu interpretieren. Dies ist die Voreinstellung.
-dtd Diese Option weist den Binding Compiler an, die übergebene Datei als eine DTD zu interpretieren.
-version Ausgabe der Versionsnummer

Tabelle 53.14: Optionen von xjc

53.15.3 Beschreibung

Der XML Schema Definiton Compiler generiert aus einer XML-Grammmatik Java-Klassen, die der Datenstruktur der Quelldatei entsprechen. Das darauf aufbauende Verfahren zur Erzeugung konkreter Objekte aus einem XML-Dokument nennt sich Unmarshalling und wird in Abschnitt 43.2.4 beschrieben.
