
  Java - Aufgabe 3b         
  -----------------
  
  Erstelle die Java Applikation "rechner.java",  
  um einen einfachen Taschenrechner  
  mit grafischer Benutzeroberfläche 
  unter Verwendung der Bibliothek "AWT" auszuführen. 
  Die Benutzeroberfläche enthält 
  ein Textfeld (Display) für Zahleneingaben und Ergebnisdarstellung,
  die Buttons +,-,*,/ für die Rechenoperationen,
  den Button "=" für die Berechnung,
  den Button "Clear" um Display und Stack zu löschen.
  Die GUI ist ohne Designer selbst zu programmieren.
  
  Als Zusatzaufgabe kann der Taschenrechner 
  mit weiteren mathematischen Funktionen 
  (wie Wurzel, Quadrat, Sinus, .. ) erweitert werden.
