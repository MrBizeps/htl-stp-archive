
Java - Projektaufgaben : 

Die Aufgaben sind einzeln auszuarbeiten 
und als vollständiger Dateiordner samt Dokumentation abzugeben. 
Die Dokumentation wird über die Vorlage "Projekt_Blatt.doc" erstellt und enthält 
	Aufgabenstellung, Programmbeschreibung, UML-Klassendiagramm (zu selbsterstellten Klassen) , Benutzerhandbuch     
Die Entwicklung ist unter Netbeans auszuführen und als Ergebnis ein ausführbares *.jar File zu erstellen.
	
Projektaufgaben - 

1) Funktionsplotter  	  
   Die Java-Hilfsklasse "plotter" soll mathematische Funktionen y(x) grafisch darstellen.   
   Über eine Auswahlliste können dazu mathematische Funktionen ausgewählt werden.
   Die Diagrammdarstellung kann entweder mit wählbarer Skalierung für x- und y-Bereich 
   oder mit einer "Autoskalierung" erfolgen. 

2) Signalanalyse 
   Einlesen von Messdaten aus einer Datei, 
   mit grafischer Darstellung der Messdaten und 
   Berechnung und Darstellung von Mittelwert, Uss, RMS.       
 
3) Direct Digital Synthesizer (DDS)
   Ein digitaler Signalgenerator für Sinus- Rechteck- Dreieck- Signale 
   mit grafischer Benutzeroberfläche und grafischer Signal Darstellung
   und Ausgabe auf CSV Datei.

4) Visualisierung komplexer Übertragungsfunktionen 
   grafische Darstellung zu Amplituden- und Phasengang von komplexen Übertragungsfunktionen. 
   Die Übertragungsfunktion wird in der allgemeinen Form : H(jw) = ( a0+a1*jw+a2*(jw)² )/( b0+b1*jw+b2*(jw)²) 
   über die Koeffizienten ( a0,a1, ... , b2 ) beschrieben. 
   
5) Ortskurven 
   grafische Darstellung der Ortskurve von komplexen Übertragungsfunktionen.
   Die Übertragungsfunktion wird in der allgemeinen Form : H(jw) = ( a0+a1*jw+a2*(jw)² )/( b0+b1*jw+b2*(jw)²) 
   über die Koeffizienten ( a0,a1, ... , b2 ) beschrieben. 
   
6) komplexe Zeigerdarstellung  
   Eine eigene Java-Hilfsklasse "komplex" soll die komplexe Rechnung ausführen 
   und die Operationen "+","-","*","/","==" sollen über Methoden implementiert werden. 
   Über eine grafische Benutzeroberfläche kann die Eingabe und Berechnung 
   von komplexen Termen erfolgen. 
   Die komplexen Zahlen werden dabei zusätzlich grafisch als Zeigerdiagramm dargestellt.    

7) Fourieranalyse    
   numerische Berechnung der diskreten Fourier Transformation (DFT)  
   eines Zeitsignals (Array) und grafische Darstellung des Amplitudenspektrums.    
   
8) Sprungantwort    
   Die Sprungantwort verschiedener RC-Schaltungen soll berechnet und 
   grafisch dargestellt werden. 
   Die Auswahl der Schaltung  (RC Tiefpass, RC Hochpass , RC mit Lastwiderstand, ... )  
   soll über eine Auswahlliste erfolgen.    

9) Impedanz     
   Die Impedanz von RLC-Schaltungen soll berechnet und 
   grafisch als Betrag- und Phasendiagramm über einen Frequenzbereich dargestellt werden. 
   
   
10) Netzwerk Analyser 
   Computer Netzwerkverbindungstest ( mit ping ) zu einer Liste von Hosts 
   und Darstellung der Ergbnisse.   
 
11) Linux User Group Administration 
    Die Benutzer und Gruppenverwaltung für das Betriebssystem Linux  
	soll über ein GUI Programm erfolgen. 
    Die Linux Konfigurationsdaten sind in den Dateien /etc/passwd und /etc/group gespeichert.  	
    
 
12) "Bilder" Archiv : 
   Archivdaten zu Bilddateien sollen über ein GUI Programm verarbeitet werden.
   Die Archivdaten sollen folgende Datenfelder aufweisen : 
   	  Bildtitel  Beschreibung  Verzeichnis  Dateiname   
   Erstellen, Ändern und Löschen von Archiveinträgen soll über dei grafische Benutzoberfläche erfolgen.  
  
  
13) WAV Player  
   Musik (Wave)-Dateien sollen geöffnet, gelesen und abgespielt werden. 
   Die Musikdaten (Samples) können für ein wählbares Zeitfenster dargestellt,  
   abgespielt und als WAVE Datei gespeichert werden. 

14) Telefonbuch 
   Name, Adresse, Telefonnummer(n) u. Handynummer werden auf einer Textdatei gespeichert. 
   Über eine Menüauswahl können neue Einträge, Änderungen  
   sowie die Abfrage u. Suche nach Telefonnummern ausgeführt werden. 

15) Wörterbuch  
   In zwei getrennten Dateien für Deutsch-Englisch u. Englisch-Deutsch 
   sind zeilenweise die Vokabel im Textformat folgend eingetragen : 
      Vokabel : Übersetzung (Zeilenende)  
   Über die Menüauswahl "Wörterbuch bearbeiten" kann ein neuer Eintrag ins Wörterbuch oder 
   das Bearbeiten vorhandener Einträge durchgeführt werden. 
   Über den Menüpunkt   "Deutsch->Englisch" kann das "Deutsch auf Englisch" Wörterbuch und 
   über den Menüpunkt   "English->German"   kann das "English to German" Dictionary abgefragt werden. 
   
16) Notenkatalog   
   Die Daten eines Notenkatalogs sollen erfasst und ausgewertet werden.
   Jeder Gegenstand einer Klasse wird dazu in einer eigenen Datei gespeichert.
   Die Dateinamen werden nach folgendem Format "Noten_Klasse_Gegenstand.dat" erstellt.
   Ein Datensatz enthält die Datenfelder : 
    	Datum, Stunde, Lehrer, Test_Note, MA_Note, Muendliche_Note   
   Über eine geeignete Benutzeroberfläche können Noteneinträge wie Notenänderungen  
   sowie die Ermittlung des Gesamtnotenstands berechnet, angezeigt und ausgedruckt werden.    

17) elektronische  Anwesenheitsliste   
   das Programm soll die Daten von Anwesenden erfassen, in einer Datei speichern
   und Listen nach bestimmten Sortierkriterien /( wie Name, Adresse, Alter, ...) erstellen. 
   Die Listen können dann als PDF-Datei exportiert werden. 
   
18) Einnahmen-Ausgaben-Buchhaltung    
   Eine Einnahmen/Ausgaben Buchhaltung soll über eine Dateiverarbeitung abgewickelt werden. 
   Ein Datensatz enthält dazu folgende Datenfelder : 
   	  lfd.Nummer  Datum  Bezeichnung  Name  Betrag  
   Einträge können hinzugefügt und bearbeitet werden, weiters kann die Abfrage von Einträgen 
   wie auch die Ausgabe des Saldos (Summe) über einen bestimmten Zeitraum durchgeführt werden. 
   
19) Vokabeltrainer
   Eine einfache und wirksame Methode, Vokabeln zu lernen, 
   besteht darin, das Karteikastenprinzip anzuwenden. 
   Vokabel befinden sich in einem von zwei Karteikästen. 
   Zu Beginn des Lernvorgangs befinden sich alle Karten im Kasten eins.
   Nun werden die Karten aus diesem Kasten nacheinander abgefragt. 
   Bei einer richtigen Antwort landet die betreffende Karte 
   in Kasten zwei, bei einer falschen Antwort verbleibt sie in Kasten eins.

20) Quiz 
   Quizfragen mit den richtigen Antworten werden in einer (verschlüsselten) Textdatei gespeichert.
   Über das Programm "quizadmin" können Quizfragen erstellt und bearbeitet werden.  
   Über das Programm "quiz" kann das Quizspiel gestartet werden. 
   Die Fragen mit der gewünschten Anzahl werden dann zufällig aus der Quizdatei gezogen, 
   die Antworten überprüft und  evtl. auch die Antwortzeiten gestoppt.  
 
21) Wettbewerb 
   Die Teilnehmerdaten und Ergebnisdaten eines Wettbewerbs werden über eine Datei verarbeitet.  
   Über eine Menüauswahl kann die Erfassung, die Eingabe der Ergebnisse 
   und dann die Ermittlung und der Ausdruck der Rangliste durchgeführt werden.
 
22) Kontoführung :  
   Das Konto für Ein- und Ausgaben eines Betriebes werden über eine Datei verarbeitet.
   Ein Datensatz enthält dazu folgende Datenfelder : 
   	  Datum   Beschreibung   Kontostelle   Betrag  
   Die Kontobewegungen (Ein-Ausgänge) und die Abfrage des Kontostandes 
   können über eine grafische Benutzeroberfläche durchgeführt werden.  
   
23) elektronische Beurteilungskatalog :  
   Die Daten eines Notenkatalogs sollen über Binärdateien verarbeitet werden. 
   Jede Klasse und jeder Gegenstand wird dazu in einer eigenen Datei gespeichert.
   Die Dateinamen werden nach folgendem Format "Noten_Klasse_Gegenstand.dat" erstellt.
   Ein Datensatz enthält die Datenfelder : 
    	Datum Stunde Lehrer Test_Note  MA_Note  Muendliche_Note   
   Über eine Menüauswahl können Noteneinträge u. Änderungen  
   und die Ermittlung des Gesamtnotenstand berechnet, angezeigt
   und als Textdatei ausgegeben werden. 

24) Arbeitskalender 
   Die Führung eines Arbeitskalenders wird über eine Textdatei durchgeführt. 
   Die Einträge haben folgendes Format :  Datum, Arbeitszeitbeginn, Dauer, Arbeitstitel,  Bearbeiter 
   Über eine Menüauswahl kann ein neuer Eintrag, sowie die Stundenabrechnung 
   für einen bestimmten Zeitraum und einen bestimmten Arbeitstitel erfolgen.  
   
25) Mitgliederverwaltung    
   Die Daten von Vereinsmitgliedern soll über eine Dateiverarbeitung abgewickelt werden. 
   Ein Datensatz enthält dazu folgende Datenfelder : 
   	  lfd.Nummer  Name  Adresse  Telefonnr  Funktion  Eintrittsdatum    
   Mitglieder können hinzugefügt und bearbeitet werden, die Suche nach Mitgliedern 
   über verschiedene Suchkriterien (wie Name oder Funktion, ...) soll durchführbar sein. 
   
26) Lotterie 6aus45
   Das Lotto-Spiel "6aus45" wird als Computerspiel ausgeführt.
   Die Daten für den Ziehungstermin, Teilnehmerliste mit den Tipps der Teilnehmer 
   sowie die Lotto-Gewinnzahlen werden über Dateien verarbeitet. 
   
27) Zeichenbrett   
   Mit der Maus kann in der Zeichenfläche gezeichnet werden. 
   Folgende Zeichenoperationen sind ausführbar : 
   Auswahl der Zeichenfarbe, der Strichstärke, Radieren, Linienziehen.
   Die Farbauswahl wird über ein Farbdialogfenster durchgeführt.
  
28) Memory
   Man soll die zusammenpassenden Kartenpaare finden, die der Computer zufällig 
   am 10x10 Spielfeld verdeckt verteilt. 
   Die Karten sind über 50 verschiedene Symbole gekennzeichnet.  
   Die Anzahl der Versuche und der gefundenen Kartenpaare wird dabei mitgezählt 
   und ausgegeben. 
 
29) Roulette 
   Ein Roulette für mehrere Spieler soll als GUI Programm erstellt werden.
