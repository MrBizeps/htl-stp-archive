
  Java - Aufgabe 5      
  -----------------
  
  Erweitere das Java Projekt "SignalAnalyse"
  mit einer Grafikausgabe zur Signaldarstellung 
  durch Einbinden und Verwendung der Klasse "Scope".  

   
  Die Auswahl der Signalform und die Eingabe der Signalparameter erfolgt über die Konsole, 
  die Diagrammdarstellung des Signals, wie auch die numerische Ausgabe der Analysewerte 
  ( "Digital Readout" ) im Grafikfenster.

  Zur Umsetzung der Ausgabe der Analysewerte ist die Klasse "Scope"
  mit einer geeigneten Methode  "display( int x, int y, String s)" 
  für eine Stringausgabe an der Position x,y zu ergänzen.
