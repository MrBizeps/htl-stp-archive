
Java - Wiederholungsfragen 

Kap. 1) bis 5) siehe Java Kurs Partl 
ab 6)  siehe Handbuch der Java Programmierung 

1) Einführung 
   Wann und von wem wurde Java entwickelt ? 
   welcher Unterschied ist zw. JavaScript,Java-Application u. Java-Applets?
   
2) Java Software 
   Was bedeutet JDK und JRE ? 
   welche Bedeutung hat eine Java Datei mit der Endung ".jar"? 
   Wie sieht der Programmaufbau einer Java-Application aus? 
   Wie werden Java Applikationen ausgeführt? 
   
3) Sprachbeschreibung 
   welche Datentypen sind in Java unterschiedlich zu C/C++ ? 
   welche Bedeutung haben Referenzen? 
   was erledigt der Garbage Collector? 
   wie werden Arrays definiert ? 
   wie werden Strings erstellt und verarbeitet? 
   nach welcher Regel erfolgt die automatische Typumformung? 
   wie kann ein Sinuswert berechnet werden? 
   was führt "System.out.println" aus? 

4) OOP in Java 
   was ist eine Klasse und was ein Objekt?    
   wie werden Klassen erstellt? 
   wie werden Instanzen erstellt?
   welche Bedeutung haben Referenzen?
   wie können Methoden der eigenen (Haupt)Klasse in static main aufgerufen werden?   
   wie werden Konstruktoren definiert?
   welche Zugriffsspezifizierer (Kapselung) sind möglich ? 
   welche Bedeutung hat "package scope"? 
   wie erfolgt die Vererbung von Klassen ? 
   wie wird der Basisklassen-Konstruktor aufgerufen? 
   wie können Interfaces verwendet werden? 
 
 
5) Fehlerbehandlung
   welche Bedeutung haben try and catch ?
   wie können Exceptions weitergereicht werden? 
   wie werden Exceptions erzeugt (geworfen) ? 

6) Graphical User Interface (GUI)  
  ( siehe Handbuch der Java Programmierung ) 
   Grafikausgabe  (Kap.24)  

   welcher Unterschied ist zw. AWT und Swing ? 
   wie wird ein GUI-Fenster erzeugt?   
   wie können grafische Grundelemente ( wie Linien, Rechtecke,...)
   dargestellt werden? 
   wie werden Farben festgelegt? 
   wie können Komponenten erstellt und sichtbar gemacht werden? 
   was sind Container Klassen?
   was erledigen die Layout Manager?
   beschreibe die Funktion eines bestimmten Layout Managers ?
   wie können Layouts verschachtelt werden? 
   wie erfolgt das Event Handling in Java?
   wie werden Listener Interfaces implementiert?
   wie können Adapter Klassen verwendet werden? 
   welche Methode hat das ActionListener Interface?   
   wie werden Listener registriert?
