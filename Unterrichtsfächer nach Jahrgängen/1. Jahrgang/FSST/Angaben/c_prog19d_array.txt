C++ - Programm : Zahlenroulette    

Erstelle das C++ Programm "roulette.cpp", 

das folgende globale Felder definiert 
	const int N=10;
	int  a[N],b[N];

zwei Spieler (a u. b) können insgesamt 10 Zahlen zwischen 0 und 36 setzen.	
( dem Feld a und b werden insgesamt 10 unterschiedliche Zufallszahlen  
  im Bereich von 0 bis 36 zugewiesen werden ) 

danach wird die Roulette-Kugel geworfen und die Gewinnzahl ermittelt
( Zufallszahl zw. 0 bis 36)    
und der Spielgewinn für Spieler a u. b angezeigt. 
( wieviele richtige gesetzte Zahlen erreicht wurden ) 

Zusätzlich ist die größte und kleinste gesetzte Zahl
der beiden Spieler zu bestimmen und anzuzeigen.
