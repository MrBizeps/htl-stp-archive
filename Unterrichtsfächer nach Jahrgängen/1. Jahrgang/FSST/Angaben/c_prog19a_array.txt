C++ - Programm : Maximum und Minimum von Arrays   

Erstelle das C++ Programm "maxmin_Klasse_katnr.cpp",
das folgende globale Felder definiert 

	const int N=10;
	int  a[N],b[N];


im Hauptprogramm sollen dem Feld a und b unterschiedliche Zufallszahlen
im Bereich von 1 bis 100 zugewiesen werden. 

Danach sind jeweils das Maximum und Minimum der Feldwerte zu bestimmen
und die Anzahl von gleichen Werten von Feld a verglichen mit Feld b.
